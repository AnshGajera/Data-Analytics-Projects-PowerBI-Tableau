# Data Visualization - Power BI & Tableau Projects

This repository contains Power BI and Tableau workbook files created for the Data Visualization course (SEM5).

## Project Files

---

## Power BI Projects

### 1. Amazon Share Price Analysis (`Amazon Share Price Analysis.pbix`)
A Power BI dashboard analyzing Amazon's stock price trends and performance metrics.

**Purpose:** Analyze historical stock price data, trends, and key performance indicators for Amazon shares.

---

### 2. Defining Power BI Model with Sample Superstore Data (`Defining power bi model with Sample Superstore data.pbix`)
A Power BI project demonstrating data modeling techniques using the Sample Superstore dataset.

**Purpose:** Learn and demonstrate Power BI data modeling concepts including relationships, calculated columns, and measures.

---

### 3. HR Attrition Analysis (`HR Attrition Analysis.pbix`)
A Power BI dashboard analyzing employee attrition patterns and factors.

**Purpose:** Identify key factors contributing to employee attrition, analyze trends, and provide actionable insights for HR decision-making.

---

### 4. IPL Dashboard using Power BI (`IPL Dashboard using Power BI.pbix`)
An interactive dashboard analyzing Indian Premier League (IPL) cricket data.

**Purpose:** Visualize IPL statistics including team performance, player stats, match results, and tournament trends.

---

### 5. Sabre Travel Dataset (`Sabre Travel Dataset.pbix`)
A Power BI analysis of travel industry data from Sabre.

**Purpose:** Analyze travel booking patterns, trends, and insights from travel industry data.

---

### 6. Sales Analysis (`SalesAnalysis.pbix`)
A comprehensive sales analysis dashboard.

**Purpose:** Track and analyze sales performance, revenue trends, and business metrics.

---

### 7. Students Cohort Analysis (`Students Cohort Analysis.pbix`)
A Power BI dashboard performing cohort analysis on student data.

**Purpose:** Analyze student cohorts, track academic performance, and identify patterns across different student groups.

---

### 8. Supermarket Sales Data Analysis (`Supermarket sales data analysis.pbix`)
A Power BI dashboard analyzing supermarket sales data.

**Purpose:** Analyze retail sales patterns, customer behavior, product performance, and revenue metrics in a supermarket context.

---

## Tableau Projects

### 9. Google Analytics using Tableau (`Google Analytics using Tableau.twb`)
A Tableau workbook analyzing synthetic Google Analytics data.

**Data Source:** `google_analytics_synthetic.csv`

**Fields Analyzed:**
- `user_id` - Unique user identifier
- `session_count` - Number of sessions per user
- `avg_session_duration` - Average session duration
- `pageviews` - Number of page views
- `bounce_rate` - Bounce rate percentage
- `country` - User's country
- `device` - Device type used
- `revenue` - Revenue generated

**Purpose:** Analyze website traffic patterns, user behavior, and revenue metrics using Google Analytics data.

---

### 10. Superstore Sample Data Analysis (`Superstore Sample Data Analysis.twb`)
A Tableau workbook performing comprehensive analysis on retail store data.

**Data Source:** `SampleSuperstore.csv`

**Fields Analyzed:**
- `Ship Mode` - Shipping method
- `Segment` - Customer segment
- `Country` - Country of sale
- `City` - City of sale
- `State` - State of sale
- `Postal Code` - Postal code
- `Region` - Geographic region
- `Category` - Product category
- `Sub-Category` - Product sub-category
- `Sales` - Sales amount
- `Quantity` - Quantity sold
- `Discount` - Discount applied
- `Profit` - Profit earned

**Purpose:** Analyze sales performance, profitability, and trends across different regions, categories, and customer segments.

---

### 11. Superstore Joins in Tableau (`Superstore_Joins in Tableau.twbx`)
A packaged Tableau workbook demonstrating data joining techniques.

**File Type:** `.twbx` (Packaged workbook with embedded data)

**Purpose:** Demonstrates various join operations and data blending techniques in Tableau.

## Software Requirements

### For Power BI Files (.pbix)
- **Microsoft Power BI Desktop** (Free download from Microsoft)

### For Tableau Files (.twb, .twbx)
- **Tableau Desktop** (2025.2 or later recommended)
- Or **Tableau Public** (free version)

## How to Use

### Power BI Files
1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
2. Open the `.pbix` files directly with Power BI Desktop
3. Explore the reports, dashboards, and data models

### Tableau Files
1. Download the `.twb` or `.twbx` files
2. Open them with Tableau Desktop or Tableau Public
3. For `.twb` files, ensure the corresponding data source files (`.csv`) are available in the expected location
4. Explore the visualizations and dashboards

## File Types

### Power BI
| Extension | Description |
|-----------|-------------|
| `.pbix` | Power BI Desktop file containing reports, data models, and queries |

### Tableau
| Extension | Description |
|-----------|-------------|
| `.twb` | Tableau Workbook - references external data sources |
| `.twbx` | Tableau Packaged Workbook - includes data within the file |
